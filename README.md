🎓 Student Management System

📘 A simple Student Management System built using Node.js, Express, and MongoDB.
It supports CRUD operations on student data.

⚙️ Features

➕ Add a new student

📋 View all students

🔍 Find student by ID

✏️ Update student details

❌ Delete student

🗂️ Student Properties

👤 Name (String)

🎂 Age (Number)

📖 Course (String)
